Objetivos

Objetivos del Programa de Software Libre:


  1. Reutilizar equipos antiguos o de bajas prestaciones
  2. Alternativa a Windows 7/8 (MS deja de dar soporte a WXP)
  3. Evitar el "pirateo" de Software privativo
  4. Reducir los costes de licencias de Software
  5. Fomentar el uso del Software Libre en Educación
  6. Poder Inventariar todo el Software y Hardware
  7. Facilitar labores de Gestión de Software
notas

Funcionalidades Añadidas

Resumen de posibilidades ofrecidas por Migasfree+Vitalinux:

  • Gestión Sencilla y Versátil del Software
  • Control Total de los Escritorios de los Alumnos
  • Carpetas compartidas Profesor - Alumnos
  • Autenticación Centralizada de Usuarios / Perfil Móvil
  • Scripts de Nautilus (plugins del explorador de archivos)
  • Control Automático del Apagado de los Equipos
  • Todo el Software necesario esta en los Repositorios
  • La mayoría de los Drivers estan disponibles
  • Normalización de los Equipos
  • Posibilidad de Congelación parcial o total del equipo
  • Puede distribuirse el Software de manera libre
